# eCommerce Apparel Store

Full stack eCommerce application using a MERN stack.
Loosely based on Udemey course by Brad Traversy

-   https://www.udemy.com/course/mern-ecommerce/?couponCode=SKILLS4SALEA

## Development

### Install

```
 npm install --dev .
 cd ui
 npm install --dev .
 cd ..
```

### Launch

```
npm run dev
```
